Os 4 paradigmas explicados

🎯

Declarativo

Você diz o quê quer. SQL, HTML, CSS. Código que se lê como uma intenção.

95% match médio
🧩

Orientado a objetos

Modela o mundo com entidades. Reutiliza, herda, encapsula. Python, Java, Kotlin.

88% match médio
🔄

Funcional

Funções puras, sem efeitos colaterais. Dados fluindo por transformações. Elixir, Haskell.

72% match médio
⚙️

Imperativo

Controle total, passo a passo. C, Go, Bash. Você decide cada instrução.

45% match médio
